Shared goal state for agents, tools, and teams.

GoalOS stores goals, dependencies, priorities, permissions, and status in one machine-readable graph so every connected system can reason against the same current state.

Open docs View GitHub

What is included

  • TypeScript core library
  • Python SDK
  • CLI for local graph workflows
  • MCP server for client integrations
  • OpenAPI and schema references in GitHub
Quick example 
import { IntentGraphClass } from '@goalos/core';

const graph = IntentGraphClass.create('team@abteex.ai', 'Launch board');

graph.addGoal({
  title: 'Ship public docs',
  status: 'active',
  priority: { level: 'critical' },
  domain: 'product',
});
