    Package Surfaces

    Everything you need to wire GoalOS into a real stack.

    GoalOS is packaged for application code, automation pipelines, terminal workflows, MCP clients, and custom integrations.

    TypeScript core

    @goalos/core

    Use the intent graph engine directly inside product code, orchestration services, or server-side workflows.

    npm install @goalos/core
    Python SDK

    goalos

    Mirror GoalOS primitives in Python with typed models and API-friendly graph operations.

    pip install goalos
    CLI

    goalos

    Create, inspect, prioritize, validate, and serve your graph from the terminal.

    npm install -g goalos
    MCP server

    @goalos/mcp-server

    Expose GoalOS as a structured context provider to Claude Desktop and any MCP-compatible client.

    npm install -g @goalos/mcp-server
    OpenAPI + spec

    spec/openapi.yaml

    Bring GoalOS into custom GPT actions, orchestrators, and HTTP-native integrations.

    spec/openapi.yaml
