Quickstart

Get an intent graph running in minutes.

GoalOS ships as a TypeScript core, CLI, MCP server, and Python SDK. Start with the package that matches your stack, then connect the rest when you need cross-tool intent sharing.

If you are evaluating the repo locally, serve the repository root so `/website/...` links resolve correctly.

Install

npm install @goalos/core pnpm add -D @goalos/mcp-server pip install goalos
1. Initialize

Run npx goalos init to create your first graph and default config.

2. Add intent

Define goals, priorities, and permissions in code or through the CLI.

3. Serve context

Start the MCP server so AI clients can read and act on the same intent graph.

4. Integrate

Use the Python SDK, LangChain, or CrewAI adapters for your existing stack.

Minimal example

import { IntentGraph } from '@goalos/core'; const graph = IntentGraph.create('launch', 'Q1 goals'); graph.addGoal({ title: 'Ship product', priority: { level: 'critical', score: 95 } });
